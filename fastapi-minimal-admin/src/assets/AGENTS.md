# assets

Static asset components — SVG icons and illustrations as React components. Same as `next-ts/src/assets/`.

## For AI Agents

- Import as React components: `import { MyIcon } from 'src/assets/icons/...'`.
- For static images (JPG/PNG), use `public/` instead.
